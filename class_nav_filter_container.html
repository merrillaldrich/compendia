Detailed Description

Tag container that displays the active filter tags and accepts tag drops to add filters.

Extends WelcomeHintContainer with drag-and-drop support so the user can drag a tag from the library area and drop it here to activate it as a file-list filter. Before any folder is loaded, a dismissible welcome hint is shown; dropping a tag onto it reveals the container and applies the tag as a filter.
