Detailed Description

Two-stack undo/redo manager using full model snapshots.

Before each user mutation, call checkpoint() to push the current model state onto the undo stack. undo() pops the most recent snapshot and restores it; redo() reapplies the last undone state. New checkpoints clear the redo stack.

Exposes QAction helpers compatible with the standard Edit-menu pattern.
