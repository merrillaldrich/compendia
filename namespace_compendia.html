◆ CacheFolderName

const char* Compendia::CacheFolderName = ".compendia_cache"
constexpr

Name of the hidden cache folder created alongside media files.

The leading dot hides the folder on Linux/macOS. Both IconGenerator and ExifParser write into this directory; MainWindow checks for its existence before loading a folder for the first time.

◆ FaceDetectionThreshold

double Compendia::FaceDetectionThreshold = 0.0
constexpr

Default HOG detector adjust_threshold for face detection.

Higher values require stronger evidence before declaring a face, reducing false positives at the cost of potentially missing smaller faces. Passed as the third argument to dlib::frontal_face_detector::operator().

int Compendia::MaxAutoFacesPerImage = 5
constexpr

Maximum number of auto-detected faces tagged per image during a recognition sweep.

Prevents crowd or large-group photos from flooding the library with dozens of auto-generated tags that the user is unlikely to label or care about.

◆ RectWarmupDelayMs

int Compendia::RectWarmupDelayMs = 3000
constexpr

Milliseconds of idle time after the last rect adjustment before the background face-cache warmup is triggered.

Increase to give the user more time to adjust before the expensive inference starts; decrease for faster cache warming at the cost of potentially warming an intermediate rect.
