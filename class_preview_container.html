Detailed Description

Widget that displays a full-size image preview inside a zoomable graphics view.

Wraps a ZoomableGraphicsView and a QGraphicsScene. The preview can be loaded from a file path or directly from a QImage. The freshen() method re-fits the image to the viewport when the containing pane is resized.
