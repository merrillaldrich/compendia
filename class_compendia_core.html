Detailed Description

Central controller that owns all application data and business logic.

CompendiaCore manages the file list model, the tag library, and coordinates asynchronous thumbnail generation with a timer-driven flush mechanism. MainWindow holds a single instance of this class and delegates all data operations to it.

void CompendiaCore::writeVisibleFileMetadata ( )

Writes JSON sidecar files for every visible (proxy-filtered) file with unsaved changes.
