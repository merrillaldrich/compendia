Detailed Description

An in-memory snapshot of all mutable model state.

Captures the tag library (families + tags + colour indices) and per-file tag assignments and ratings. EXIF data, pHash values, file paths, and filter state are excluded — they are either read-only or view state and do not need to be undone.
