Detailed Description

The application's main window, wiring UI widgets to LuminismCore operations.

MainWindow owns the generated Ui::MainWindow and a LuminismCore instance. It does not own any file or tag data directly; all data lives in core.

◆ updateFolderStatsLabel()

void MainWindow::updateFolderStatsLabel ( )

Updates the persistent folder-stats label in the status bar.

Shows the number of images, videos, folders, and total file size for the currently loaded folder. Files inside .luminism_cache directories are excluded. Clears the label when no files are loaded.
