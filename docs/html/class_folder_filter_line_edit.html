Detailed Description

A QLineEdit with shell-style path autocomplete for the folder filter.

As the user types a path fragment, the widget searches the folder paths of all loaded files and suggests the characters that follow the typed text up to the next path delimiter, picking the alphabetically first candidate.

Accepting a suggestion works the same way as shell tab-completion:

  • Right arrow or Tab: accept and keep focus in this field
  • Enter/Return: accept and keep focus
  • Backspace/Delete: dismiss the suggestion
