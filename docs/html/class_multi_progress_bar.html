Detailed Description

A status-bar widget that tracks multiple concurrent background processes and cycles the display between them when more than one is active.

Each process is identified by the Process enum. Callers register a process with startProcess(), advance it with increment(), and optionally terminate it early with finishProcess(). When two processes are active simultaneously the label and bar alternate on a configurable timer.
