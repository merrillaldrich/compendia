Detailed Description

A QGraphicsView subclass that supports mouse-wheel zoom centred on the cursor.

Scrolling the mouse wheel zooms in or out while keeping the point under the cursor stationary. Zooming out further than the viewport size is prevented so the image cannot be shrunk to nothing.
