Detailed Description

Intermediate base class that adds a dismissible welcome hint label to TagContainer.

Manages an external hint label placed in the parent splitter or layout alongside the scroll area. The hint is shown grayed-out until activateWelcome() is called, then becomes interactive (click-to-dismiss and/or drop-to-dismiss depending on which flags are set by the subclass). On dismissal the scroll area is revealed and welcomeDismissed() is emitted.

Subclasses call setAcceptsClickToDismiss() and/or setAcceptsDropToDismiss() in their constructors to opt in to each behaviour.
