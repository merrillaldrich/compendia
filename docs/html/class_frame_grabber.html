Detailed Description

Captures a representative frame from each video file in a batch.

Operates entirely on the main thread using an async state machine so that the QMediaPlayer it creates shares the same COM/WMF apartment as every other QMediaPlayer in the application, avoiding apartment-model conflicts on Windows.

Call grab() to begin a batch. Results, progress ticks, and the final summary arrive as signal emissions on the main thread. The class is single-use per batch: create a new instance for each grab operation and call deleteLater() when finished() fires.
