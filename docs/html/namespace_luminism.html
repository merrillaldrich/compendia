◆ CacheFolderName

const char* Luminism::CacheFolderName = ".luminism_cache"
constexpr

Name of the hidden cache folder created alongside media files.

The leading dot hides the folder on Linux/macOS. Both IconGenerator and ExifParser write into this directory; MainWindow checks for its existence before loading a folder for the first time.

◆ FaceDetectionThreshold

double Luminism::FaceDetectionThreshold = 0.0
constexpr

Default HOG detector adjust_threshold for face detection.

Higher values require stronger evidence before declaring a face, reducing false positives at the cost of potentially missing smaller faces. Passed as the third argument to dlib::frontal_face_detector::operator().
