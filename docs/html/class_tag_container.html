Detailed Description

Base container widget that displays a set of tags grouped into TagFamilyWidget items.

TagContainer manages a FlowLayout of TagFamilyWidget children, each of which holds one or more TagWidget children. Subclasses (NavFilterContainer, NavLibraryContainer, TagAssignmentContainer) add drag-and-drop or mouse interaction behaviour on top of this base.

◆ filter()

void TagContainer::filter ( const QString & text)

Shows only tag families and tags whose name starts with text.

Shows only families and tags whose tag name contains text (case-insensitive); shows all when empty.

When text is empty all tags and families are shown. Otherwise each TagWidget is shown only if its tag name contains text (case-insensitive), and a TagFamilyWidget is shown only when at least one of its child TagWidgets is visible.

Parameters
textThe prefix to filter on.
