Detailed Description

Graphics item that draws a rounded rectangle overlay and supports edge/corner dragging.

When constructed with interactive = true the item accepts hover events and left-button press/move/release events. Hovering near any of the eight edge or corner handles changes the cursor; dragging one of those handles resizes the rect live. On mouse release rectChanged is emitted with the new scene-coordinate rect.

The corner radius and pen width are expressed in screen pixels (cosmetic), so they remain constant at any zoom level.
