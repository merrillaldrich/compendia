Detailed Description

Detects and recognises faces across images using dlib's HOG detector and ResNet embeddings.

Converts QImages to the dlib pixel format, runs the HOG-based frontal face detector, aligns detected faces via a 5-point shape predictor, and computes 128-d embeddings using a pre-trained ResNet model. Results can be cached to disk to avoid redundant computation on repeated scans.

◆ computeEmbeddingFromRegion()

dlib::matrix< float, 0, 1 > FaceRecognizer::computeEmbeddingFromRegion ( const QImage & img,
const QRectF & normalizedRect )

Computes a face embedding for a user-drawn region in img.

Runs the HOG detector and picks the detection with the best IoU against normalizedRect. Falls back to a synthetic rectangle from normalizedRect if no detection has IoU > 0.3.

Parameters
imgThe source image.
normalizedRectThe user-drawn region in normalised image coordinates (0.0–1.0).
Returns
A 128-d embedding, or an empty matrix if the embedding could not be computed.
