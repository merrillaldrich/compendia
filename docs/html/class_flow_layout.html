Detailed Description

A custom QLayout that arranges child widgets in a wrapping flow, like inline text.

Items are placed left-to-right and wrap to the next row when the available width is exhausted. The layout supports both a parent-widget form and a standalone form, and exposes independent horizontal and vertical spacing.
