Detailed Description

Base widget class that carries a base colour for tag and tag-family widgets.

TagWidget and TagFamilyWidget both inherit from this class so that child widgets such as TagWidgetCloseButton can retrieve the parent's colour without a dynamic cast to a specific subclass.
