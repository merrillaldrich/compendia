Detailed Description

A QListView subclass that accepts tag drops to apply tags to media files.

Displays the filtered list of media files and handles drag-and-drop of tag widgets onto file items. When a tag is dropped onto a selected item, it is applied to all currently selected files; when dropped onto an unselected item it is applied only to that one file.
